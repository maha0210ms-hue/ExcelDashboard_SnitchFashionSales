## ExcelDashboard_SnitchFashionSales
The objective of this project is to analyze Snitch Fashionâ€™s sales performance data to identify key revenue drivers, customer purchasing patterns, and product category trends. The project aims to uncover insights related to sales growth, profitability, discount impact, and regional performance in order to support data-driven business decisions and improve overall operational efficiency.
## Application Used
MS Excel 2021
## Steps Followed
- Importing raw data(Snitch_Fashion_Sales.csv) into Excel Workbook.
- Data Cleaning and Transformation - Used IF(),AVERAGEIFS(),DATE(),DATEVALUE(), basic mathematical formulae and power query for handling missing and inconsistent values.
- Data Visualization - generated Pivot table from the cleaned dataset, used charts(Bar, Line and Pie) and trendline for analyzing the growth trend.
- Dashboard Creation - Used all the Charts and filters to create an interactive dashboard.
## Insights Derived
The Fashion store should concentrate on fixing affordable price margin with standardized discounts based on the Cities will increase the units of the products sold which in-turn yields the higher revenue.
